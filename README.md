# AI Translator API

Веб-приложение для перевода текста с использованием нейросетей Hugging Face (модели Helsinki-NLP/opus-mt).
Реализовано на FastAPI, PostgreSQL, Docker. Включает веб-интерфейс для удобной демонстрации.

## 🚀 Быстрый запуск

   ```bash
   git clone <ссылка_на_репо>
   cd ai-translator-api
